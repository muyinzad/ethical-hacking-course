# 🛡️ Ethical Hacking Course - Setup Guide

## Quick Start

### Step 1: Set Up Your Lab

**Required Software:**
- VirtualBox (free) or VMware (paid)
- 8GB+ RAM recommended

**Download:**
1. Kali Linux: https://www.kali.org/get-kali/
2. Metasploitable: https://information.rapid7.com/download-metasploitable

---

### Step 2: Install Kali Linux

```
1. Download Kali ISO
2. Create VM in VirtualBox
3. Assign 2GB+ RAM
4. Assign 20GB+ storage
5. Install OS
6. Update: apt update && apt upgrade
```

---

### Step 3: Network Configuration

```
VirtualBox Network Settings:
├── NAT (Internet access)
├── Host-Only (Lab network)
└── Bridged (Direct network access)

For lab: Host-Only between Kali & Metasploitable
```
